# AgentFlow Media Kit

This folder contains the current public-facing AgentFlow logo assets based on the same shared brand lockup used by the landing page and chat header.

Canonical logo names:
- `agentflow-logo-primary.svg`
- `agentflow-logo-badge.svg`

Supporting legacy/public assets already in `/public`:
- `agentflow-logo.svg`
- `agentflow-logo-square.png`

Brand notes:
- Primary gold: `#f2ca50`
- Gold press: `#d4af37`
- Background: `#101010`
- Surface background: `#131313`
- Primary text: `#ffffff`
- Secondary text / badge F: `rgba(255,255,255,0.85)`
- Badge style: rounded dark square, gold `A`, soft white `F`, short gold underline
- Wordmark style: `Agent` in white, `Flow` in gold

Usage:
- Use `agentflow-logo-primary.svg` for headers, landing pages, docs, and press references.
- Use `agentflow-logo-badge.svg` when only the square `AF` mark is needed.
- Prefer dark backgrounds for the current lockup treatment.

- Primary logo filename: `agentflow-logo-primary.svg`
